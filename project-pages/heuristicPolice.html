Project Objective

In this project I applied heuristics of user interface design to identify usability problems for the Toronto Police Service website. I used the Usability Heuristics recommended by the Nelson Norman Group as a tool for the analysis of the website.

The resulting 34 page report included a comprehensive analysis of the website. It cited specific heuristics critera that aren't being met and includes recommendation and mockups that make the website more functional and user friendly. Here are snipbits from the report:
